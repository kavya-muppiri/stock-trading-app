# User Flow

## Stock Trading Application User Flow

```
Start
  │
  ▼
Open Application
  │
  ▼
Register (New User)
  │
  ▼
Login
  │
  ▼
Dashboard
  │
  ├── View Portfolio
  │
  ├── Search Stocks
  │
  ├── Add to Watchlist
  │
  ├── Buy Stocks
  │
  ├── Sell Stocks
  │
  ├── View Transaction History
  │
  ├── View Leaderboard
  │
  ├── Update Profile
  │
  └── Logout
  │
  ▼
End
```

---

## Description

1. User opens the application.
2. New users register for an account.
3. Existing users log in.
4. After successful login, the dashboard is displayed.
5. Users can:
   - View portfolio
   - Search stocks
   - Buy stocks
   - Sell stocks
   - Manage watchlist
   - View transaction history
   - View leaderboard
   - Update profile
6. User logs out to end the session.

## Conclusion

This user flow represents the navigation path of the Stock Trading Application from registration to logout.
